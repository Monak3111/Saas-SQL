# SaaS Business Intelligence Analytics Dashboard

## Overview

This project simulates a SaaS company's data environment using MySQL and analyzes key business metrics through SQL queries and an interactive Power BI dashboard.

## Features

* Customer, subscription, payment, product usage, support ticket, and marketing campaign tracking
* SaaS KPI analysis including:

  * Monthly Recurring Revenue (MRR)
  * Annual Recurring Revenue (ARR)
  * Churn Rate
  * Customer Lifetime Value (CLV)
  * Customer Acquisition Cost (CAC)
  * Daily & Monthly Active Users (DAU/MAU)
* Cohort Analysis and Revenue Trend Analysis
* Interactive Power BI Dashboard for business insights

## Tech Stack

* MySQL
* SQL
* Power BI

## Key Learning Outcomes

* Database Design & Data Modeling
* Advanced SQL (JOINs, CTEs, Subqueries, Window Functions)
* Business Intelligence & KPI Reporting
* Data Visualization and Dashboard Development
